# CurrencyConverter
A simple Python program which uses an API to calculate currency conversions based off of user input.

Go to https://freecurrencyapi.com to attain a free API key for converting.

Create an .env file and paste in `API_KEY=\<your_api_key\>`

To run the program, open the terminal and navigate to the CurrencyConverter folder using `cd CurrencyConverter`.

To install the necessary packages, run `pip install -r requirements.txt`

To the program, enter `python3 converter.py`  into the terminal.

When prompted, enter the acronym of the currency you want to convert from.

Enter the acronym of the currency you want to convert to.

Enter the amount you want to convert.

You should see your output printed in the terminal.
